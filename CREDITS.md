## Parallax Engine

Parallax Engine is a derivative work based on the NEXO Engine, enhanced with VR capabilities and additional features.

**Current Development:**

**VR Implementation (2026):**

- Stereo rendering pipeline
- Motion controller support
- VR locomotion and interaction systems

**Engine Enhancements (2026):**
- RMB-gated camera controls
- PBR and toon shading systems

### NEXO Engine Core Features

The following foundational systems were inherited from NEXO Engine:

- **Entity-Component-System (ECS) Architecture** - Modern, data-oriented design
- **Rendering Pipeline** - OpenGL-based 3D rendering with multi-camera support
- **Physics Integration** - Jolt Physics engine integration
- **Asset Management System** - Texture, model, and material loading
- **Scene Management** - Hierarchical scene graph and entity management
- **Editor Framework** - ImGui-based editor interface and docking system
- **Scripting Support** - C# scripting via .NET SDK integration
- **Build System** - CMake with vcpkg dependency management
- **Cross-Platform Support** - Windows and Linux compatibility

### NEXO Engine Documentation and Resources

The NEXO Engine team provided excellent documentation and resources:
- Comprehensive build system with CMake presets
- Automated testing with gtest
- CI/CD pipelines with GitHub Actions
- Code quality monitoring with SonarCloud
- Extensive documentation website
